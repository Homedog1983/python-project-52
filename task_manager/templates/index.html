    Итоговый тренировочный проект: "Менеджер задач"

    Профессия: "python-developer"

    Образовательная платформа: Хекслет. Практические курсы по программированию

    Проект отрабатывает навыки, в числе которых:

    • Установка и настройка бекэнд-фреймворка Django, подключение базы данных PostgreSQL
    • Максимальное использование встроенных классов Django (отображения, формы и т.д.). Наследование и/или использование миксинов при усложнении поведения встроенных классов.
    • Создания CRUD сущностей (отдельное приложение, модель, форма, отображения) с различными связями между ними (o2m, m2m) с использованием Django ORM
    • Различные поведения компонентов CRUD в зависимости от аутентификации пользователя
    • Создание шаблонов совместно с фронтенд-фреймворком bootstrap
    • Фильтрация вывода из БД по различным критериям при отображении списка сущностей
    • Создание автотестов, подключение коллектора ошибок, деплой

    {% translate "Особенности проекта"%}:

    Без входа доступна только страница с выводом всех зарегистрированных пользователей

    После регистрации и входа открываются новые вкладки "Статусы", "Метки", "Задачи"

    Эти сущности может создавать, изменять и удалять только залогиненный пользователь

    Самая интересная вкладка - Задачи. При создании задачи используется статус (выбирается из существующих). Примечательно, что как и при создании, так и при изменении задачи, можно добавить метки и назначить исполнителя (выбираются из существующих).

    На странице задач есть фильтр, который выводит задачи в зависимости от заданных параметров (статус, метки, исполнитель). Также можно вывести только свои задачи.
